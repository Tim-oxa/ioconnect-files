## 2. Какие данные мы обрабатываем
2.1. **Данные Telegram:**
- Telegram ID пользователя;
- username (если передаётся Telegram и доступен Администратору);
- служебные данные взаимодействия с Ботом (например, команды/нажатия кнопок), необходимые для работы Сервиса.

2.2. **Технические данные использования Сервиса:**
- user-agent (строка идентификации клиентского ПО/устройства, если передаётся при обращении к компонентам Сервиса);
- объём трафика (агрегированные показатели использования).

2.3. **Платёжные данные:**
- обработка оплаты осуществляется через платёжного провайдера (в т.ч. **moneta.ru**);
- Администратор может получать сведения о факте оплаты (статус, сумма, дата/время, идентификатор транзакции);
- Администратор не хранит полные реквизиты банковских карт.

## 3. Цели обработки
Мы обрабатываем данные для:
- предоставления доступа к функциям Сервиса и учёта Тарифа/лимитов;
- аутентификации/связывания пользователя с учётной записью (по Telegram ID);
- обеспечения безопасности и предотвращения злоупотреблений (антифрод);
- обработки обращений в поддержку;
- выполнения обязанностей, предусмотренных законодательством РФ (в применимой части).

---

## 4. Правовые основания обработки (РФ)
Обработка осуществляется на основании:
- заключения и исполнения договора (Пользовательское соглашение);
- согласия пользователя — в случаях, когда оно требуется;
- законных интересов Оператора (безопасность, предотвращение злоупотреблений) — в пределах, допускаемых законом.

---

## 5. Сроки хранения
5.1. Данные хранятся **не дольше, чем это необходимо** для целей обработки.

5.2. Ориентиры хранения (если не требуется иное по закону):
- Telegram ID и сведения об оплаченных периодах/статусах доступа — на период использования Сервиса и разумный срок для поддержки и разрешения споров;
- user-agent и объём трафика — на период действия доступа и разумный срок для антифрода/диагностики.

5.3. Пользователь может обратиться с запросом об удалении данных через @wis_tim. Удаление возможно, если это не противоречит требованиям закона и необходимости защиты прав Оператора (например, при споре по оплате).
